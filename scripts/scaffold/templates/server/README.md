# Server Template

This directory should contain the template files for the server workspace.

To populate, run:
```bash
bun run generate:scaffold-templates
```

Or manually place the baseline server files here preserving the directory structure:
- `apps/server/src/index.ts`
- `apps/server/src/config.ts`
- `apps/server/src/middleware/auth.ts`
- `apps/server/src/middleware/error.ts`
- `apps/server/package.json`
